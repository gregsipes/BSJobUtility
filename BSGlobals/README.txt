Job Information

    Description: 	The intention of this project is to provide code that can be shared amongst the different types of projects,
					from the SQL jobs to the UI projects. This project includes file IO, data IO, SMTP, and FTP/SFTP code. Additionally,
					formatting type specific code, like turning 56.10- into a known .NET format can be found in this project.
